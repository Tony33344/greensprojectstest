📋 How to Use These Templates

For Supabase: Go to Authentication → Email Templates and paste the HTML. Variables like {{ .Token }} and {{ .ConfirmationURL }} are already formatted for Supabase.

For Hostpoint.ch (SMTP): Use a template engine to replace variables with actual data, then send via SMTP with HTML content type.

Preview: Click any template above to see it in your browser. Use browser DevTools to test mobile responsiveness.
